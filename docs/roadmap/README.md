# 中長期ロードマップ（設計サンプル保管）

> **本ディレクトリの位置付け**
>
> 実装着手が 1 年以上先となる中長期テーマの設計サンプルを保管する。
> 選考フェーズでは **実装済みテーマ（01〜05）と検証証跡を主軸に提示する** ため、
> 2026-07 に [server-monitor 改善設計](../server-monitor-improvements/README.md) の
> 一次導線から本ディレクトリへ縮退した。設計を破棄したのではなく、
> 実装フェーズが近づいた時点で一次導線へ戻す。

---

## 縮退の判断理由（2026-07）

- 一次導線に 17 本の設計書が並ぶと、実装済みのテーマと「設計のみ」のテーマの区別が
  つきにくく、量が先に立って個々の完成度が伝わりにくい。
- 実装着手が 1 年以上先のテーマは、選考時点では「考えられること」の証明にはなるが
  「動かしたこと」の証明にはならない。露出の主軸を実装済み + 証跡へ寄せる。
- この判断自体も [ADR](../adr/README.md) と同じ思想で記録し、後から経緯を辿れるようにする。
